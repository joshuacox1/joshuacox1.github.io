Solving a linear system of congruences in general

This solver solves for \(x\) all systems of congruences

\[\begin{aligned} a_1x&\equiv b_1 \pmod{m_1}\\ &\vdots\\ a_nx&\equiv b_n \pmod{m_n} \end{aligned}\]

for \(a_i,b_i,m_i,n\in \mathbb{Z}, m_i, n\geq 1\). If there is a solution, it will be a congruence \(x=S \pmod{M}\), which the solver will return. If there is no solution, the solver will report which equations are unsolvable or irreconcilable.

(1) \(x\equiv \;\) \(\;(\text{mod }\)\()\)
(2) \(x\equiv \;\) \(\;(\text{mod }\)\()\)
(3) \(x\equiv \;\) \(\;(\text{mod }\)\()\)
(4) \(x\equiv \;\) \(\;(\text{mod }\)\()\)

Solution

How the solver works

The solver first attempts to solve each congruence \(ax\equiv b \pmod{m}\) individually. Let \(d=\gcd(a,m)\). If \(d \nmid b\), the congruence has no solutions and hence the whole system has no solutions. Otherwise, let \(a'=a/d\), \(b'=b/d\) and \(m'=m/d\). Now \(a'\) is a unit mod \(m'\) and the solutions \(x\) are exactly those integers satisfying

\[ x \equiv b'(a')^{-1} \pmod{m'}\text{.} \]

If all equations can be individually solved in the form \(x=c_i \pmod{m'_i}\), the solver continues. Consider a pair of equations \(x=c_i \pmod{m'_i}\) and \(x=c_j \pmod{m'_j}\) and let \(g=\gcd(m'_i,m'_j)\). If \(c_i \not\equiv c_j \pmod{g}\), the equation has no solutions. Otherwise, let \(m^\ast_i=m'_i/g\), \(m^\ast_j=m'_j/g\), and use the extended Euclidean algorithm to find integers \(p,q\) satisfying \(m_i^\ast p + m_j^\ast q = 1\). Then let \(e=c_im_j^\ast q + c_jm_i^\ast p\) and \(\ell=\operatorname{lcm}(m'_i,m'_j)\). The solutions are then exactly those integers \(x\) of the form

\[ x\equiv e \pmod{\ell}\text{.} \]

In general, the system of equations \(x\equiv c_i \pmod{m'_i}\) has a solution if and only if \(c_i\equiv c_j \pmod{\gcd(m'_i,m'_j)}\) for all pairs of equations indexed \(i,j\). If this is satisfied, then the overall solution may be obtained by iterating the process for pairs on successive elements. In particular the modulus is \(\operatorname{lcm}(m_1',\ldots,m_n')\). This is what the solver does.
